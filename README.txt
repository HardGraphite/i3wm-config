I3-WM CONFIGURATION
===================

Dependencies
------------

X             -- of course
i3-wm         -- the window manager itself
picom         -- X11 compositor
polybar       -- status bar
dunst         -- notification daemon
clipsim       -- clipboard manager (https://github.com/lucas-mior/clipsim)
rofi          -- multifunction list menu
feh           -- tool to set wallpaper
xsetroot      -- X11 root window tool
fcitx5        -- input method
xsecurelock   -- screen lock
xss-lock      -- auto lock

Suggested tools
---------------

NetworkManage -- network management
udisks2       -- USB disks management
